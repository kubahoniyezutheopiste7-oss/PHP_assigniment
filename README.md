---

## Short Description
The Hostel Booking Management System is a web-based application developed to manage hostel rooms and bookings efficiently.  
The system allows an administrator to view available rooms, manage bookings, and monitor upcoming stays through a user-friendly dashboard.  
It is designed to simplify hostel management using PHP, MySQL, and a clean dashboard interface.

---

## Features
-  Admin login and logout
-  Dashboard summary:
  - Total number of rooms
  - Total bookings
  - Upcoming stays
-  Room management
-  Booking management
-  View upcoming bookings with:
  - Guest name
  - Room number
  - Check-in date
  - Check-out date
- Secure database access using prepared statements
-  Simple and responsive user interface

---

## Technologies Used
- **Frontend:** HTML, CSS, Bootstrap
- **Backend:** PHP
- **Database:** MySQL
- **Server:** Apache (XAMPP)
